## Installation

1. Set up a .env file using the .envEXAMPLE as a template
2. In the command line type "npm i" to install all dependencies
3. Login to mysql then run source db/schema.sql
4. In the command line type "npm run seed" to seed database
5. Finally in the command line type "npm start" and open insomnia to test the app
